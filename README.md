# Mihomo Config

*Mihomo 内核* 覆写文件。

## 简介

该项目提供了一个覆写配置文件，符合 *Mihomo 内核* 规则，可用于任何使用 *Mihomo 内核* 的代理客户端。

## 使用方式

复制 **config_next.yaml** 的 *raw* 地址，导入到 *代理客户端* 的**覆写**选项中。
